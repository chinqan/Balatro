## 1. 極端狀況推演

### 1.1 數值破局推演 (Game + Level)

| 場景 | 推演內容 | 驗證標準 |
|------|---------|---------|
| 最強構築 vs 金注 Floor 8 | 5 個 ×DMG 遺物 + 皇家同花順 + Lv.10 牌型 | 必須可以在 3 回合內擊敗 Boss |
| 最弱開局 vs Floor 3 | 無神器、僅基礎牌型 | 必須有至少 1 條活路 |
| 無限循環檢測 | 重觸發遺物 + 重觸發封印 | 連鎖次數硬上限 = 50 |
| 經濟破產 | 連續 3 場 0 收入 | 必須仍能通過 Floor Boss |

### 1.2 效能壓力測試 (Tech + Audio)

| 場景 | 同屏元素 | 目標 FPS |
|------|---------|:-------:|
| 5 張牌連鎖 + 5 遺物觸發 | ~100 粒子 + 10 數字 + 5 Shader 切換 | ≥ 60 |
| 20 連擊 + 全屏 CRT | ~300 粒子 + 20 數字 + 8 Shader | ≥ 55 |
| Boss 毀滅動畫（全特效） | ~500 粒子 + 火焰 Shader + 慢動作 | ≥ 30 |

### 1.3 認知負荷評估 (UX)

| 場景 | 同時資訊量 | 建議處理 |
|------|:--------:|---------|
| 普通出牌 | 3 層（牌型+面值+增強） | ✅ 可接受 |
| 5 遺物連鎖 | 7 層（上述+5遺物效果） | ⚠️ 需延遲動畫展示 |
| 超長連鎖 20+ | 10+ 層 | 🔴 自動加速按鈕 |

## 2. 無障礙規格

| 功能 | 規格 |
|------|------|
| 色弱模式 | ATK/DMG/Shield 改用形狀標記（圓/方/三角）+ 圖案底紋 |
| 減少動態 | 關閉粒子、CRT Shader、畫面震動，保留核心數字動畫 |
| 行動端退化 | 粒子數 ÷ 2、停用 CRT、降低 Shader 複雜度 |
| Safari 音頻 | 首次觸碰時啟動 AudioContext（iOS 限制） |
| 文字大小 | 最小 14px，支援 UI 縮放 1.0x / 1.5x / 2.0x |

### 3.2 術語表 (Glossary)

| 術語 | 定義 |
|------|------|
| Base ATK | 基礎攻擊力，對應 Balatro 的 Chips |
| DMG Mult | 傷害倍率，對應 Balatro 的 Mult |
| Relic | 遺物/神器，對應小丑牌 (Joker) |
| Scroll | 奧術捲軸，對應塔羅牌 (Tarot) |
| Elixir | 星辰靈藥，對應星球牌 (Planet) |
| Pact | 虛空契約，對應幻靈牌 (Spectral) |
| Blessing | 永久加持，對應代金券 (Voucher) |
| Floor | 地城樓層，對應底注 (Ante) |
| Run | 一局完整遊戲流程 |
| Seed | 隨機種子，控制所有隨機事件 |
