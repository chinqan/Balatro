## 1. 局外解鎖系統 (Meta-Progression)

### 1.1 解鎖層級

| 內容類型 | 初始可用 | 需解鎖 | 解鎖方式 |
|---------|:-------:|:-----:|---------|
| 賭注難度 | 白注 | 紅~金注(7個) | 以前一級通關即解鎖下一級 |
| 起始牌組 | 3 種 | 12+ 種 | 各有獨立解鎖條件 |
| 挑戰模式 | 0 | 20 種 | 以白注通關解鎖前 5 個，後續漸進 |
| 神器 | ~80 | ~70 | 達成特定遊戲里程碑 |
| 消耗品 | 全部 | — | 無需解鎖 |

### 1.2 解鎖條件範例

| 解鎖目標 | 條件 |
|---------|------|
| 紅注難度 | 以白注通關 1 次 |
| 「亡靈牌組」 | 使用契約 10 次 |
| 挑戰「禁神器」 | 以任意賭注通關 3 次 |
| 傳奇神器「終焉之眼」 | 以金注通關 |
| 「煉金牌組」 | 單次傷害超過 10,000 |

- 主選單「收藏庫」按鈕顯示全局進度百分比

## 4. 平衡性測試框架

### 4.1 數值模擬規範

```
模擬腳本規格：
- 輸入：種子範圍 (1-10000)、賭注等級、牌組
- 過程：自動進行完整 Run（隨機選牌+購買策略）
- 輸出：勝率、平均通關樓層、平均傷害曲線、經濟曲線
```

### 4.2 破局檢測閾值

| 指標 | 正常範圍 | 破局閾值 | 處理方式 |
|------|---------|---------|---------|
| 白注勝率 | 30-50% | > 70% | 調低基礎數值 |
| 金注勝率 | 5-15% | > 30% | 增加高階限制 |
| 單次最高傷害 | ≤ Boss HP ×3 | > Boss HP ×10 | 檢查乘法遺物 |
| 平均通關時間 | 20-40 分鐘 | < 10 分鐘 | 增加樓層or Boss HP |

### 4.3 回歸測試流程

```
每次數值調整後：
1. 執行 1000 次模擬（白注+標準牌組）
2. 執行 1000 次模擬（金注+各牌組）
3. 比對前後勝率/傷害曲線
4. 若偏移 > 5%，標記為需審查
```
